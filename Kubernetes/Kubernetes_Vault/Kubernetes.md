#### A. What is Kubernetes
(+) Kubernetes is a tool for automated management of containerized applications also known as a container orchestration tool
==> It is used to automate application infrastructure and make it easy to manage

#### B. Kubeadm, Kubelet, and Kubectl
(+) `Kubeadm`: is a tool which automate a large portion of the process of setting up a cluster. It will make our job much easier
(+) `Kubelet`: is the essential component of Kubernetes that handles running containers on a node. Every server that will be running container needs Kubelet
(+) `Kubectl`: Command-line tool for interacting with the cluster once iti s up. We will use this to manage the cluster
